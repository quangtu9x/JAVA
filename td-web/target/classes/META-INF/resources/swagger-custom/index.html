Recommended Test Flow

  1. POST /api/v1/documents
  2. GET /api/v1/documents
  3. GET /api/v1/documents/{id}
  4. PUT /api/v1/documents/{id}
  5. POST /api/v1/documents/search
  6. POST /api/v1/documents/deleted/search
  7. DELETE /api/v1/documents/{id}
  8. DELETE /api/v1/documents/{id}/permanent (ADMIN)
